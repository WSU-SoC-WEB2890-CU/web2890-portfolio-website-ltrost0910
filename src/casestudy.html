Case Study: Salazar Restaurant

An overview of my role, challenges, and the results of this project.

Overview

Context: Personal Portfolio Project

Role: Frontend Developer

Tools: Bootstrap, SCSS, JavaScript

Problem Statement

For my final project in WEB 2620 - Advanced CSS during Spring 2023, I was tasked with developing a 6-page website that met the following criteria:

  • The website must be visually appealing and professionally designed.
  • Use a provided webpack-based starter project.
  • Utilize either Bootstrap or Bulma for layout.
  • Implement a mobile-first design, ensuring responsiveness across at least three breakpoints (small, medium, and large).
  • Style the site using Sass.
  • Employ Flexbox or CSS Grid for layout management.
  • The site must include an index.html landing page plus at least five additional pages with clear links between them, as well as at least one form. All pages should maintain a cohesive look and feel.
  • The site must feature 10+ components, including but not limited to:
    • Open-sourced images, navbars, menus (e.g., navs/tabs), media objects, buttons, alert messages, cards, forms, lists, tables, links, and jumbotrons.
  • All components should be customized using Sass to ensure a unique design.
  • The final project should be deployed on Netlify and submitted to GitHub.

Solution

To meet the project requirements, I created a mobile-first, responsive website with a clean, professional design. The website was built using Bootstrap for consistent layout management and Sass for efficient styling. I implemented a modular SCSS architecture to keep the code maintainable and scalable.

The website includes six pages: Home, Food, Directions, News, About Us, and Contact Us. I used Bootstrap's responsive classes along with custom media queries to ensure a seamless experience across various screen sizes.

Additionally, I created a Figma prototype of the website, which can still be accessed.

Page Descriptions

  • Home: Introduces the restaurant and provides a menu structure for exploring the other pages. It offers an engaging starting point for visitors to navigate the site.
  • Food: Displays the lunch and dinner menus, with a visually appealing layout to highlight the variety of offerings.
  • Directions: Features an embedded Google Map of the restaurant’s location, along with essential details like the address and operating hours.
  • News: Showcases images and links to media coverage, including reviews of the chef and the restaurant.
  • About Us: Focuses on the owner/chef, detailing their culinary background and the food philosophy, complemented by social media links and photos.
  • Contact Us: Includes a sign-up form for the restaurant's newsletter, encouraging visitors to stay connected.

Outcome

The site successfully achieved its goals, offering a visually stunning and user-friendly experience on both desktop and mobile devices. The website’s responsive design ensures that it adapts smoothly to varying screen sizes, delivering optimal performance across devices.

Reflection

One of the main challenges was ensuring the site’s performance with a large number of high-quality images. By optimizing assets and using lazy loading techniques, I achieved fast load times across all devices.
